📧 Spam Email Detection
🚀 Overview

A machine learning project that classifies emails as Spam or Ham using NLP techniques and TF-IDF vectorization.

🛠️ Tech Stack
Python
Pandas, NumPy
Scikit-learn
NLTK
⚙️ Workflow
Data cleaning & preprocessing
Text vectorization (TF-IDF)
Model training (Naive Bayes / Logistic Regression)
Evaluation & prediction
📊 Result

Achieved ~95% accuracy in spam classification.

▶️ Run
git clone <repo-link>
cd spam-email-detection
pip install -r requirements.txt
